## üîπ Module Overview
Module 4 focuses on interactive computing environments that enable data scientists to document experiments, analyze results, and share reproducible workflows. The module introduces Jupyter Notebook and JupyterLab as core tools for exploratory data analysis and experimentation.

Learners gain insight into how notebook sessions operate, how different kernels are used to support multiple programming languages, and how the underlying Jupyter architecture is structured. The module also explores the Anaconda-based Jupyter ecosystem and provides an overview of cloud-hosted Jupyter environments and their capabilities for data science work.

> This module highlights the role of notebooks as a central tool for experimentation, collaboration, and reproducible data science.

## üéØ Learning Objectives
By completing this module, learners will be able to:

1. Describe how notebooks are used within JupyterLab.  
2. Explain how to work effectively in a notebook session.  
3. Describe the core components of the Jupyter architecture.  
4. Explain the role of kernels and how they are used in notebooks.  
5. Identify key tools available in Anaconda-based Jupyter environments.  
6. Describe cloud-based Jupyter environments and their data science features.
