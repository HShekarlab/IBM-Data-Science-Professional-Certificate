## üîπ Module Overview
Module 5 combines statistical programming with collaborative software development practices. The module introduces R as a powerful language for data analysis and manipulation, along with the RStudio environment used to develop and run R code efficiently. Learners explore data visualization concepts in R and create graphical outputs using built-in plotting capabilities.

In addition, the module covers distributed version control concepts and focuses on Git and GitHub as essential tools for modern data science workflows. Learners gain both conceptual understanding and hands-on experience with repositories, commits, branches, pull requests, and collaborative development practices, culminating in a practical project.

> This module bridges data analysis and collaboration by integrating programming, visualization, and version control skills.

## üéØ Learning Objectives
By completing this module, learners will be able to:

1. Describe the capabilities of R and the RStudio development environment.  
2. Create visualizations using built-in plotting functions in R.  
3. Explain version control concepts and the roles of Git and GitHub.  
4. Describe the purpose of source code repositories and how GitHub supports them.  
5. Explain how to create a GitHub account and set up a repository.  
6. Demonstrate how to edit, upload, and manage files on GitHub.  
7. Explain the purpose of branches and how changes are merged within workflows.
