## üîπ Module Overview
Module 3 introduces core software components and resources commonly used in data science workflows. Learners explore widely used libraries across scientific computing, visualization, machine learning, and deep learning, and gain a practical understanding of how these libraries support model development.

The module also explains how APIs function within data science systems, with a focus on REST-based request and response patterns. In addition, learners examine open data resources and discover how pre-built machine learning models can be leveraged to solve real-world problems using model repositories.

> This module connects tooling, data resources, and models into a cohesive view of applied data science.

---

## üéØ Learning Objectives
By completing this module, learners will be able to:

1. Identify the key tasks involved in building a data science model.  
2. Recognize common libraries used for scientific computing, visualization, machine learning, and deep learning.  
3. Define a REST API and explain the relationship between requests and responses.  
4. Describe datasets and common sources of data.  
5. Explore open datasets available through data exchange platforms.  
6. Explain how machine learning models can be applied to solve practical problems.  
7. Navigate a model exchange platform to discover and use existing models.
