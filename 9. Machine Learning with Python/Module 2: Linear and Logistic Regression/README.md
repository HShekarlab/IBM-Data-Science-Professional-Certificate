## üîπ Module Overview
Module 2 introduces two key regression techniques in machine learning: linear and logistic regression. Learners explore how regression models are applied to predict continuous outcomes and understand the differences between simple and multiple linear regression. The module also demonstrates the use of polynomial and non-linear regression for capturing complex patterns in data.

In addition, logistic regression is introduced as a method for classification problems. Practical exercises using scikit-learn enable learners to train and test both regression and classification models on real-world datasets. A cheat sheet summarizing formulas and key concepts supports hands-on learning.

> This module equips learners with essential regression skills needed for predictive modeling in machine learning workflows.

---

## üéØ Learning Objectives
By completing this module, learners will be able to:

1. Explain the purpose of regression analysis for predicting continuous outcomes.  
2. Describe simple linear regression and when to apply it.  
3. Apply simple linear regression using scikit-learn on real datasets.  
4. Differentiate multiple linear regression from simple linear regression.  
5. Apply multiple linear regression using scikit-learn on real datasets.  
6. Interpret polynomial and non-linear regression models for complex relationships.  
7. Explain logistic regression for classification and its differences from linear regression.  
8. Apply logistic regression using scikit-learn for classification tasks.
