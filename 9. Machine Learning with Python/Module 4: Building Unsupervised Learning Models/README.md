## üîπ Module Overview
Module 4 introduces unsupervised learning techniques to discover hidden patterns in unlabeled data. Learners explore clustering methods, including K-Means for customer segmentation, and advanced density-based models such as DBSCAN and HDBSCAN for identifying clusters in spatial datasets.

The module also covers dimensionality reduction methods to simplify high-dimensional datasets. Techniques such as Principal Component Analysis (PCA), t-SNE, and UMAP are applied to interpret and visualize complex data structures. A supporting cheat sheet provides key methods, practical use cases, and comparisons for unsupervised learning.

> This module equips learners with the ability to uncover patterns, segment data, and reduce dimensionality for effective analysis in unsupervised learning scenarios.

## üéØ Learning Objectives
By completing this module, learners will be able to:

1. Describe how unsupervised learning identifies patterns in unlabeled data.  
2. Explain the concept of clustering and its applications.  
3. Apply K-Means clustering using scikit-learn for customer segmentation.  
4. Compare DBSCAN and HDBSCAN clustering methods and their outcomes.  
5. Explain the purpose of dimensionality reduction for complex datasets.  
6. Apply Principal Component Analysis (PCA) to reduce data dimensions and interpret patterns.  
7. Use t-SNE and UMAP for visualizing high-dimensional data.
